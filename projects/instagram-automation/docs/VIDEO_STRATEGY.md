# 비디오 생성 전략

**목적**: 릴스 제작 비용 최적화 및 단계별 품질 향상  
**기준**: 팔로워 수 기반 단계별 전략

---

## 📊 3단계 전략 개요

### 선택 기준 A: 빠른 제작 (Phase 1, 현재)
**대상**: 팔로워 0~100명  
**방식**: T2I 이미지 슬라이드쇼 → ffmpeg 영상 변환  
**비용**: ~$0.3 per 릴스 (이미지 3~5장)  
**소요 시간**: 약 5분 per 릴스

### 선택 기준 C: 하이브리드 (Phase 2)
**대상**: 팔로워 100~500명  
**방식**: 훅 씬(첫 3초)만 Kling v3.0 Pro, 나머지는 A 방식  
**비용**: ~$2~3 per 릴스  
**소요 시간**: 약 35분 per 릴스

### 선택 기준 B: 고퀄리티 (Phase 3)
**대상**: 팔로워 500+ 이후  
**방식**: 모든 씬 Kling v3.0 I2V로 실제 영상 생성  
**비용**: ~$6~10 per 릴스  
**소요 시간**: 약 1.5~2시간 per 릴스

## 🎯 선택 기준 A: 빠른 제작 (현재)

### 워크플로우

```
1. Nanogen T2I 이미지 생성 (3~5장)
   ↓
2. ffmpeg 슬라이드쇼 변환
   - 각 이미지 1~2초 표시
   - Ken Burns 효과 (줌/팬)
   - 전환 효과 (fade, slide)
   ↓
3. 자막 오버레이 (ffmpeg)
   ↓
4. BGM 추가 (선택)
   ↓
5. 최종 릴스 (5~10초)
```

### ffmpeg 명령어 예시

```bash
# 1. 이미지 슬라이드쇼 생성 (Ken Burns 효과)
ffmpeg -loop 1 -t 2 -i frame_01.png \
       -vf "zoompan=z='min(zoom+0.0015,1.5)':d=50:s=1080x1920" \
       frame_01_zoom.mp4

# 2. 슬라이드 연결 (크로스페이드)
ffmpeg -i frame_01_zoom.mp4 -i frame_02_zoom.mp4 \
       -filter_complex "[0][1]xfade=transition=fade:duration=0.5:offset=1.5" \
       combined.mp4

# 3. 자막 추가
ffmpeg -i combined.mp4 \
       -vf "drawtext=text='봄 트렌치코트':fontsize=60:fontcolor=white:x=(w-text_w)/2:y=h-100" \
       final.mp4
```

### 장점
- ✅ 매우 빠름 (5분 이내)
- ✅ 저렴함 (~$0.3)
- ✅ Gena 캐릭터 일관성 유지
- ✅ 충분한 초기 품질

### 단점
- ⚠️  슬라이드쇼 느낌 (움직임 제한적)
- ⚠️  고급스러운 느낌 부족

## 🚀 선택 기준 C: 하이브리드 (Phase 2)

```
1. 훅 씬 (첫 3초)

   ↓
2. 나머지 씬 (3~7초)

   - 비용: ~$0.3~0.5
   ↓
3. ffmpeg 조립
   ↓
4. 자막 + BGM
   ↓
```

- **훅 씬에 비용 집중**: 첫 3초가 가장 중요
- 나머지는 빠르고 저렴하게
- 전체 비용 ~$2~3 (합리적)

### 장점
- ✅ 훅 씬 고품질 (조회수 상승 기대)
- ✅ 비용 효율적 (전체 고퀄보다 저렴)
- ✅ 제작 시간 단축 (~35분)

### 단점

## 📝 기술 노트

### ffmpeg Ken Burns 효과

```bash
# 줌 인 (1.0x → 1.5x)
zoompan=z='min(zoom+0.0015,1.5)':d=50:s=1080x1920

# 줌 아웃 (1.5x → 1.0x)
zoompan=z='max(zoom-0.0015,1.0)':d=50:s=1080x1920

# 좌→우 팬
zoompan=z='1.2':x='iw/2-(iw/zoom/2)':y='ih/2-(ih/zoom/2)':d=50:s=1080x1920
```

### 전환 효과

```bash
# 크로스페이드
xfade=transition=fade:duration=0.5:offset=1.5

# 슬라이드
xfade=transition=slideright:duration=0.5:offset=1.5

# 와이프
xfade=transition=wiperight:duration=0.5:offset=1.5
```
