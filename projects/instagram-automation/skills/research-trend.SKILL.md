## 🔧 실행 흐름

### 1단계: Instagram 탐색탭 리서치

**검색 키워드 조합**:
```
주제 + 시즌 + "룩"
예: "봄 데님 룩", "데님 스타일링"
```

**WebSearch 활용**:
```
site:instagram.com "봄 데님 룩" OR "#봄데님"
```

### 2단계: 게시물 빈도 분석

**상위 트렌드 판단 기준**:
- 최근 7일간 게시물 수
- 좋아요/저장 수 비율
- 해시태그 사용 빈도

### 3단계: 경쟁 계정 3개 선정

**선정 기준**:
- 팔로워 1만~10만 (중소형 인플루언서)
- 최근 1주일 내 활동
- 콘텐츠 스타일 유사 (패션 OOTD)

### 4단계: 스타일 패턴 추출

**분석 요소**:
- 색상 조합 (상위 3개)
- 아이템 조합 (상의+하의)
- 촬영 장소 (스튜디오/거리/카페)
- 포즈 스타일 (정면/측면/워킹)
