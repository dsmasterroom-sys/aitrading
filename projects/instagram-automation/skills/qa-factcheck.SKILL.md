**용도**: 팩트 체크 및 정보 검증  
**사용 에이전트**: qa-reviewer  
**버전**: 1.0

---

## 📥 입력

- copy.md (슬라이드 카피)
- items.json (제품 정보)

---

## 📤 출력

qa-report.md (팩트 오류 섹션):
```markdown
## 팩트 체크

### 고 심각도
- Slide 05: 가격 불일치
  - 카피: "3만원대"
  - items.json: 29,900원
  - 수정: "2만원대" 또는 "약 3만원"

### 중 심각도
- Slide 06: 브랜드명 오타
  - 카피: "뮤신사 스탠다드"
  - items.json: "무신사 스탠다드"
```

---

## 🔍 검증 항목

### 가격 정확성

#### 표기 vs 실제
```
copy.md: "3만원대 아이템"
items.json: price = 29900

검증:
- 29,900원 = 2만원대 (20,000~29,999)
- ❌ "3만원대" → **고 심각도**
```

#### 권장 표현
- "약 3만원"
- "2만원대 후반"
- "3만원 이하"

### 브랜드/제품명

#### 정확성
```
copy.md: "뮤신사 스탠다드"
items.json: brand = "무신사 스탠다드"

검증:
- ❌ 오타 → **고 심각도**
```

#### 대소문자
- 공식 표기 따름
- "무신사" ≠ "MUSINSA"

### 구매 링크 유효성

#### HTTP 응답 확인
```python
import requests
response = requests.get(items.json["purchase_url"])

if response.status_code == 200:
    # 유효
else:
    # ❌ 링크 오류 → **고 심각도**
```

#### 품절 확인
- 구매 페이지에 "품절" 문구?
- ❌ 품절 → **중 심각도**
