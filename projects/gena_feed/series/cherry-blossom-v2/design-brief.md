## 4. 슬라이드별 비주얼 기획서

### slide-01 (훅 — 텍스트+시각 충돌)
- **구조**: 풀블리드 이미지 + 상단 1/3 대형 타이포 오버레이
- **주인공 시각 요소**: 벚꽃 터널 아래, 바람에 머리 날리며 살짝 움츠린 여성 뒷모습. 오버사이즈 트렌치 자락 펄럭임
- **타이포 배치**: "벚꽃 OOTD" var(--fs-display) 72px+ 상단 좌측 비대칭. "봄옷 입으면 망한다" var(--fs-title). 장식 따옴표 손그림 포인트. var(--ff-accent) 800
- **색상 토큰**: var(--text-light), 메인에 라임/샤르트뢰즈 포인트 (--cherry-lime)
- **조명**: [Golden Hour] 역광 림라이트, 그레인 강
- **나노젠**: Yes
- **헤어**: gena_ref_06_long_wave.png
- **의상**: 오버사이즈 베이지/카멜 트렌치 + 크림 터틀넥 + 와이드팬츠
- **의상 레퍼런스**: flat-lay 생성
- **손그림**: 장식 따옴표, 하단 "swipe →" 화살표

### slide-02 (문제 제기)
- **구조**: 풀블리드 이미지 + 하단 1/3 대형 타이포 스택
- **주인공 시각 요소**: 수변 벚꽃길, 얇은 니트 입고 팔짱 낀 채 어깨 움츠린 사이드 프로필
- **타이포 배치**: 하단 좌측 비대칭. "작년의 나:" var(--fs-display) 볼드 화이트 세리프 스택. 나머지 var(--fs-body)
- **색상 토큰**: var(--text-light), 서브 opacity 80%
- **조명**: [Golden Hour] 쿨 쉬프트 — 블루-그린 시네마틱
- **나노젠**: Yes
- **헤어**: gena_ref_03_basic_straight.png
- **의상**: 얇은 크림 니트 + 라이트 카디건
- **의상 레퍼런스**: flat-lay 생성

### slide-03 (반전 팩트 — L3)
- **구조**: 풀블리드 이미지 + 센터 대형 숫자 타이포
- **주인공 시각 요소**: 한강 수변 벚꽃길 와이드샷, 바람에 꽃잎 날림. 인물 매우 작게
- **타이포 배치**: 센터 "18°C" 144px급 거대 숫자 var(--accent). 서브 우하단 비대칭 var(--fs-caption). 화면 40%+ 차지
- **색상 토큰**: "18°C" var(--accent), 서브 var(--text-light)
- **조명**: [Golden Hour] 쿨 블루 쉬프트
- **나노젠**: Yes
- **손그림**: "18°C" 옆 바람 곡선/화살표

### slide-04 (실패 공식)
- **구조**: 풀블리드 이미지 + 우하단 대형 타이포
- **주인공 시각 요소**: 해질녘 벚꽃길, 그림자 긴 골든아워. 얇은 블라우스+스커트 실루엣 (역광)
- **타이포 배치**: 우하단 비대칭. "봄옷 = 벚꽃룩?" var(--fs-display) + "완벽한 실패 공식" var(--fs-title)
- **색상 토큰**: var(--text-light), 서브 opacity 70%
- **조명**: [Golden Hour] 정통 — 드라마틱 역광 실루엣
- **나노젠**: Yes
- **헤어**: gena_ref_02_medium_straight.png
- **의상**: 화이트 블라우스 + 플리츠 스커트 (실패 예시)
- **의상 레퍼런스**: flat-lay 생성

### slide-05 (3단 레이어링 — L3)
- **구조**: 풀블리드 이미지 + 좌측 세로 타이포 스택
- **주인공 시각 요소**: 오후 햇살, 트렌치 걸친 모델 미디엄 클로즈업. 이너 레이어(터틀넥) 살짝 보임
- **타이포 배치**: 좌측 세로 계단식: "14~16°" / "7~9°" / "3~5°" var(--fs-display). 각 옆 아이템명 var(--fs-caption). 서브 var(--ff-hand) Dooneun
- **색상 토큰**: var(--text-light)
- **조명**: [Golden Hour] 웜 뉴트럴 오후 자연광
- **나노젠**: Yes
- **헤어**: gena_ref_05_updo.png
- **의상**: 트렌치코트 + 터틀넥 + 히트텍 이너 (레이어 보이는 구성)
- **의상 레퍼런스**: flat-lay 생성
- **손그림**: 온도 사이 ↓ 화살표

### slide-06 (아이템 조합)

- **색상 토큰**: var(--text-light)
- **조명**: [Golden Hour] 웜 뉴트럴
- **나노젠**: Yes
- **헤어**: gena_ref_01_long_twinhalfup_wave.png
- **의상**: 머스타드/카멜 오버사이즈 트렌치 + 아이보리 터틀넥 + 차콜 와이드팬츠 + 앵클부츠
