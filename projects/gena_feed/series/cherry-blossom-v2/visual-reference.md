# Visual Reference Analysis — cherry-blossom-v2

## 레퍼런스 이미지 분석 (사용자 제공 2장)

### 공통 디자인 언어

1. **풀블리드 사진이 캔버스** — 프레임, 카드, 보더 없음. 사진 자체가 배경
2. **대형 타이포가 비주얼의 주인공** — 텍스트가 정보 전달이 아니라 구도의 일부
3. **혼합 타입 계층** — HUGE 디스플레이 + 중간 서브 + 작은 바디 카피
4. **필름 에스테틱** — 그레인, 뮤트/시프트 컬러, 빈티지 감성
5. **비대칭·유기적 텍스트 배치** — 그리드에 갇히지 않음, 자유롭게 흐름
6. **텍스트와 이미지 겹침** — 대담하게 인물/오브제 위에 텍스트 배치
7. **세리프/이탤릭 폰트** — 감성적 임팩트, 에디토리얼 무드
8. **손그림 요소** — 화살표, 곡선, 장식 따옴표 등 수작업 느낌
9. **UI 요소 제거** — 박스, 카드, 보더, 그림자 일절 없음
10. **컬러 악센트는 타이포에만** — 이미지는 뮤트, 텍스트에 포인트 컬러

### 이미지 1 (visual+typo 2.jpg) 상세
- 웜 골든/세피아 필름 워시 — 전체에 앰버-골드 틴트
- 라이프스타일 포토 (카페 테이블, 티, 식물)
- 타이포: 볼드 세리프 이탤릭 + 디스플레이 in **샤르트뢰즈/라임 옐로우**
- 장식 따옴표("")가 디자인 요소로 활용
- 손그림 화살표가 텍스트 블록을 연결
- 어두운 영역에 텍스트 배치 (자연스러운 명암 대비 활용)

### 이미지 2 (visual+typo.jpg) 상세
- 쿨 블루-그린 시네마틱 톤 — 야간/실내 앰비언트 라이트
- 인물 풀블리드: 소파 위 캔디드 포즈, 시네마틱 프레이밍
- 타이포: 거대 볼드 화이트 세리프 "You are the only one" — 스택, 좌정렬
- 화면의 40%+ 차지하는 모뉴멘탈 타이포
- 세컨더리 텍스트 수직 스태킹 ("WHO SAW MY YESTERDAY")
- 하단 풀와이드 에디토리얼 쿼트 (소형 올캡스)
- 우상단 날짜/크레딧 텍스트
- 매거진 커버 / 에디토리얼 포스터 에스테틱

## v2 적용 방향

### 전면 폐기하는 것
- 카드형 레이아웃 (메모카드, 포스트잇)
- 분할(50/50) 인포그래픽 레이아웃
- 체크리스트/체크박스 UI 레이아웃
- CSS 일러스트/아이콘 장식
- 밝은 단색 배경 타이포 슬라이드

### 전면 채택하는 것
- **모든 슬라이드 = 풀블리드 사진 + 타이포 오버레이**
- 텍스트는 이미지 위에 직접 — 반투명 배경 최소화 (필요 시 미세 그라데이션만)
- 대형 세리프/이탤릭 타이포가 구도의 일부
- 혼합 타입 계층 (display 72px+ / sub 36px / body 28px)
- 비대칭 텍스트 배치 — 좌하단, 우상단, 대각선 등 자유롭게
- 필름 그레인 + 뮤트 톤 일관 유지
- 손그림 요소 (화살표, 곡선, 밑줄) 포인트 활용
- 포인트 컬러는 타이포에만 (샤르트뢰즈 or 브랜드 accent)
