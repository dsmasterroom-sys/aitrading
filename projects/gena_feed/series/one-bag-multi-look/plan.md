# 원백 멀티룩 (one-bag-multi-look) 기획서

## 시리즈 개요
- 주제: 하나의 가방, 세 가지 스타일 — genarchive 크로스백의 멀티 스타일링
- 유형: 캐러셀 (8장)
- 타겟: MZ세대 20-35세 여성, 패션·라이프스타일
- 협찬: genarchive 미니 크로스백 (간접 노출)
- 조명 조건: Overcast (5500-6500K) — 시크/모던 쿨톤

## 감정 곡선
```
slide-01 (표지/훅)  : 온도 ●●●●○  스크롤 멈춤 유발
slide-02 (공감)     : 온도 ●○○○○  "나도 이랬어"
slide-03 (전환)     : 온도 ●●●○○  "근데 이거 알아?"
slide-04 (정보1)    : 온도 ●●○○○  LOOK 1 캐쥬얼
slide-05 (정보2)    : 온도 ●●○○○  LOOK 2 미니멀
slide-06 (정보3)    : 온도 ●●●○○  LOOK 3 스트릿
slide-07 (증거)     : 온도 ●●●●○  제품 디테일
slide-08 (CTA)      : 온도 ●●●●●  저장·팔로우·예고
```

## 슬라이드별 기획

### slide-01 (표지/훅)
- 역할: 스크롤 멈춤
- 핵심 메시지 방향: "가방 하나로 이렇게 달라진다"는 궁금증 유발
- 레이아웃: 풀블리드 다크 배경 + 대형 타이포
- 감정 온도: ●●●●○
- 나노젠 이미지: No (텍스트 중심)
- 배경: var(--surface-dark)

### slide-02 (공감)
- 역할: 공감
- 핵심 메시지 방향: 아침마다 가방 고르느라 고민하는 공감대 형성
- 레이아웃: 풀블리드 이미지 + 하단 오버레이 텍스트
- 감정 온도: ●○○○○
- 나노젠 이미지: Yes — 거울 앞 고민 (gena_straight)
- 이미지 방향: 미니멀 침실, 거울, 침대 위 다양한 가방

### slide-03 (전환 — 소재)
- 역할: 전환
- 핵심 메시지 방향: 이 가방의 핵심은 소재와 디테일
- 레이아웃: 풀블리드 매크로 이미지 + 최소 텍스트
- 감정 온도: ●●●○○
- 나노젠 이미지: Yes — 나일론 매크로 클로즈업 (제품 참조)
- 이미지 방향: 나일론 직조 텍스처, 지퍼, 버클 극접사

### slide-04 (정보 — LOOK 1 캐쥬얼)
- 역할: 정보
- 핵심 메시지 방향: 캐쥬얼 데일리룩과 크로스백의 조화
- 레이아웃: 풀블리드 이미지 + 상단 텍스트 오버레이
- 감정 온도: ●●○○○
- 나노젠 이미지: Yes — 3단계 파이프라인 (gena_straight)
- 의상: 크림 니트 가디건 + 화이트 티 + 와이드 데님 + 스니커즈
- 가방 착용: 크로스바디 (대각선)
- 배경: 연남동 골목

### slide-05 (정보 — LOOK 2 미니멀)
- 역할: 정보
- 핵심 메시지 방향: 미니멀 오피스룩에도 어울리는 크로스백
- 레이아웃: 풀블리드 이미지 + 하단 텍스트 오버레이
- 감정 온도: ●●○○○
- 나노젠 이미지: Yes — 3단계 파이프라인 (gena_braid)
- 의상: 블랙 오버핏 블레이저 + 화이트 셔츠 + 블랙 와이드 슬랙스 + 로퍼
- 가방 착용: 숄더백 (한쪽 어깨)
- 배경: 강남 빌딩 입구

### slide-06 (정보 — LOOK 3 스트릿)
- 역할: 정보
- 핵심 메시지 방향: 스트릿 무드에서 힙색으로 변신
- 레이아웃: 풀블리드 이미지 + 상단 텍스트 오버레이
- 감정 온도: ●●●○○
- 나노젠 이미지: Yes — 3단계 파이프라인 (gena_hippie)
- 의상: 블랙 레더 봄버 재킷 + 화이트 블라우스 + 화이트 플리츠 스커트 + 탄 웨스턴 부츠
- 가방 착용: 웨이스트백 (힙색)
- 배경: 망원동 계단

### slide-07 (증거 — 제품 디테일)
- 역할: 증거
- 핵심 메시지 방향: 제품 실물 퀄리티와 디자인 디테일 확인
- 레이아웃: 풀블리드 제품 히어로샷 + 디테일 포인트 텍스트
- 감정 온도: ●●●●○
- 나노젠 이미지: Yes — 제품 히어로샷 (제품 참조만)
- 이미지 방향: 라이트 그레이 배경, 3/4 앵글, 스트랩 드레이핑

### slide-08 (CTA)
- 역할: CTA
- 핵심 메시지 방향: 저장 + 팔로우 + 다음편 예고
- 레이아웃: 타이포 중심 + 브랜드 로고
- 감정 온도: ●●●●●
- 나노젠 이미지: No (텍스트 중심)
- 배경: var(--surface-dark)
