# 스토리 제작 워크플로

## 실행 순서 (2-Gate)

```
[자동 구간 1]
1. contents-marketer → skills/plan-content.md
                     → 스토리 유형 결정:
                        - 정보형: 핵심 1가지 메시지
                        - 투표형: 선택지 2개
                        - 제품소개형: 협찬 제품 하이라이트
                        - CTA형: 캐러셀·릴스 연결 유도

2. contents-marketer → skills/write-copy.md
                     → 스토리 카피 (텍스트 최소화 원칙)

3. designer          → skills/design-brief.md
4. designer          → skills/nanogen-image.md

→ [Gate A] 사용자 중간 확인: 기획 + 카피 + 비주얼 방향

[자동 구간 2]
5. developer         → skills/nanogen-workflow.md + skills/build-html.md
                     → 출력: output/{시리즈명}/story.png

6. qa-reviewer       → skills/qa-check.md

→ [Gate B] 사용자 최종 확인: 완성 PNG + QA 리포트
```

## 기본 설정
- 캔버스: 1080×1920px
- 텍스트: 최소화 (이미지 중심)
- 연결 게시물: 해당 캐러셀/릴스 링크 스티커 추가 안내
