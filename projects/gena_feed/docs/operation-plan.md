## 3. 1일 콘텐츠 제작 사이클

하루 1건의 메인 콘텐츠를 아래 흐름으로 제작합니다.
4개 Gate에서 반드시 사용자 컨펌을 받습니다.

```
[시작] 나노젠 로컬 서버 가동
  ↓
[Gate 1] researcher → 리서치 결과 확인
  ↓ 사용자 OK
[Gate 2] contents-marketer → 기획+카피 확인
  ↓ 사용자 OK
[Gate 3] designer → 비주얼+나노젠 프롬프트 확인
  ↓ 사용자 OK
[제작] developer → 나노젠 API 호출 → HTML/영상 구현
  ↓
[QA] qa-reviewer → 검수 (HIGH/MID 시 수정 반복)
  ↓
[Gate 4] 최종 확인 → 업로드 준비 완료
```

예상 소요: Gate 응답 속도에 따라 1건당 약 30-60분
