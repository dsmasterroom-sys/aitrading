## 🎯 콘텐츠 방향

### 주제 힌트
> 없으면 비워두세요 — researcher 에이전트가 트렌드 기반으로 주제 3개를 추천합니다.

```
예: "요즘 MZ들이 쓰는 선크림 성분 이야기"
예: "직장인 봄 코디 5가지"
예: (비워두면 researcher가 자율 결정)
```

### 타깃 심리
> 이번 게시물을 보고 팔로워가 어떤 감정을 느끼길 원하나요?

- [ ] 정보: "몰랐던 걸 알게 됐다" → 저장율 ↑
- [ ] 재미: "저장·공유하고 싶다" → 확산 ↑
- [ ] 공감: "나만 이런 게 아니구나" → 팔로우 ↑
- [ ] 호기심 해결: "왜 그런지 알고 싶었어" → 댓글 ↑

### 훅 아이디어 (선택사항)
> 첫 슬라이드 표지 문구 방향. 없으면 에이전트가 제안합니다.

```
예: "피부과 원장이 집에서 쓰는 SPF50"
예: (비워두면 contents-marketer가 제안)
```

## 🎨 비주얼 방향

### 시리즈 무드
> 아래 중 하나를 선택하세요. 조명 조건이 자동으로 결정됩니다.

- [ ] **시크/모던** → [Overcast] 쿨톤 자연광
- [ ] **캐주얼/힙** → [Bright Sunny] 맑은 직사광
- [ ] **페미닌/럭셔리** → [Golden Hour] 아침 골든아워
- [ ] 미정 (designer 에이전트가 주제에 맞게 결정)

### 젠아 등장 여부
- [ ] 등장 — 무드: (페미닌 / 시크 / 캐주얼 중 선택)
- [ ] 등장 안 함 (제품·텍스트 중심)
- [ ] 미정

### 참고 레퍼런스 (선택사항)
> 비슷한 분위기의 인스타그램 게시물 URL이나 키워드

```
예: "토마스 삭스 미니멀리즘 느낌"
예: "@계정명 이 게시물 분위기"
예: (없으면 brand-identity 기본 무드 적용)
```

## 🚀 Claude Code 실행 방법

이 파일을 저장한 뒤, `gena-feed/` 폴더에서 Claude Code를 열고 아래처럼 입력하세요:

```
이 기획서를 바탕으로 캐러셀 콘텐츠 만들어줘.
workflows/carousel.md 워크플로대로 진행해줘.

[brief.md 내용 붙여넣기 또는 파일 첨부]
```

Gate 1(리서치 확인) → Gate 2(기획+카피 확인) → Gate 3(비주얼 확인) → Gate 4(최종 확인)
순서대로 사용자 확인을 받으면서 진행됩니다.
