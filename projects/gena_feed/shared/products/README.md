# shared/products/
# 협찬 제품 레퍼런스 이미지를 제품별 폴더로 관리합니다.
#
# 현재 등록:
#   genarchive_crossbag/   ← @genarchive.kr 미니크로스백 이미지 2종
#
# 신규 협찬 추가 방법:
#   1. shared/products/{브랜드명_제품명}/ 폴더 생성
#   2. 제품 이미지 저장 (정면·디테일컷 최소 2장)
#   3. product-info.md 작성 (아래 템플릿 참조)
#   4. AGENT.md의 designer contextFiles에 자동 포함됨
#
# product-info.md 템플릿:
#   - 제품 기본 정보 (브랜드, 제품명, 가격, 소재, 컬러, 사이즈)
#   - 마켓링크 (쿠팡·스마트스토어 등 — 있으면 반드시 포함)
#   - 협찬 조건 (필수 멘션, 포함 요소, 금지 사항, 기한)
#   - 셀링 포인트 / 타깃 소구 포인트
#
# 마켓링크 활용 규칙:
#   product-info.md에 마켓링크가 있으면 researcher가 해당 페이지를
#   스크래핑하여 부족한 제품 정보(상세 스펙, 소재 설명, 리뷰 키워드,
#   배송 정보 등)를 수집하고 product-info.md를 보완한다.
#   → 오케스트레이터가 보완된 내용을 사용자에게 확인 후 저장
