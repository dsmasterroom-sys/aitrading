## 검증된 캐러셀 패턴

| 패턴명 | 훅 공식 | 나노젠 기능 | 저장율 |
|--------|---------|------------|--------|
| 숫자 리스트형 | "피부과 원장이 쓰는 5가지" | Text-to-Image | 높음 |
| 비포/애프터 | "2주 전 vs 지금" | Garment Swap / Identity Swap | 매우 높음 |
| 스타일링 비교 | "이 원피스 3가지 무드" | Garment Swap × 3 | 매우 높음 |
| 루틴 공개 | "내 아침 10분 루틴 공개" | Text-to-Image + I2V | 높음 |
| 오해 vs 진실 | "다들 잘못 알고 있는 OO" | Text-to-Image | 매우 높음 |
| 가격대별 추천 | "1만원대도 이게 있어?" | Text-to-Image | 높음 |
| 시네마틱 무드 | (텍스트 없이 비주얼로 승부) | Text-to-Video (Veo) | 중간 |
| 제품 쇼케이스 | "협찬 | 이거 써봤는데..." | I2V (Kling v3, zoom_in) | 높음 |
| 트렌드 해석 | "요즘 다들 이거 쓰는 진짜 이유" | Workflow Studio | 중간 |

## 레이아웃 패턴 (developer 참조용)

| 레이아웃명 | 구조 | 적합 슬라이드 |
|-----------|------|-------------|
| 풀블리드 | 이미지 전체 + 텍스트 오버레이 | 표지·CTA |
| 분할 (50/50) | 좌: 이미지 / 우: 텍스트 | 비교·정보 |
| 카드형 | 흰 카드 + 배경색 | 리스트·증거 |
| 타이포 중심 | 텍스트가 주인공 | 전환·강조 |
| 그리드 | 2×2 또는 3×1 분할 | 루틴·단계 |

## 레이아웃 제약
- 같은 레이아웃 2연속 사용 금지
- 같은 감정 온도 3연속 금지
