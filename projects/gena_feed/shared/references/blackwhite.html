LAYOUT TEST
기본형
제품 사진 중심 / 정보 분산 / 가격만 강조
  • • 정보 탐색 피로가 크다
  • • 브랜드 무드 전달이 약하다
  • • 저장·공유 포인트가 부족하다
편집형
헤드라인 / 근거 / 디테일 컷 / CTA가 한 흐름으로 연결된다
  • • 읽는 순서와 기억 포인트가 분명하다
  • • 무드와 기능 설명을 동시에 담는다
  • • 구매 전환용 후속 스토리 설계가 쉽다
호감 포인트 / 이탈 포인트
좋아하는 이유
  • • 차분한 화면이 브랜드 신뢰를 높인다
  • • 한 컷 한 메시지가 기억에 남는다
  • • 디테일 설명이 구매를 돕는다
떠나는 이유
  • • 정보가 너무 많아 화면이 무겁다
  • • 제품보다 그래픽이 먼저 보인다
  • • 가격과 이유가 연결되지 않는다
