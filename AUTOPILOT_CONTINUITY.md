# AUTOPILOT_CONTINUITY

## 목적
자율 실행 중 작업 연속성 단절 방지 + 문제해결 루프 자동화.

## 공통 제어 루프 (모든 프로젝트 공통)
1. PLAN: 목표/완료조건/타임박스(15분)
2. DO: 실행
3. CHECK: 결과/오류코드/진척률 기록
4. ACT: 수정 후 즉시 재실행

## 강제 규칙
- 5분 무진척: 중간보고
- 15분 무진척: 우회전략 자동 전환
- 동일 오류 2회 반복: 원인분석 모드로 전환 후 재설계
- 정상 루틴은 무소음, 실패/차단만 보고

## 증거 기반 보고 포맷
- 변경 파일
- 실행 명령
- 실행 결과 요약 (성공/실패 + 원인)
- 산출물 경로

## 금지
- 같은 재시도만 반복하고 원인 미분석 상태 유지
- "진행 중"만 반복 보고
