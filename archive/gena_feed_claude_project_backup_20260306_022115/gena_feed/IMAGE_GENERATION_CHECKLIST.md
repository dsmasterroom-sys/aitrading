# 이미지 생성 체크리스트 (필수)

**⚠️ 이미지 생성 = 과금 발생 → 반드시 사전 확인 필수**

## 생성 전 필수 체크

### 1단계: 범위 확인
- [ ] **몇 개 게시물?** (무조건 1개만!)
- [ ] **총 몇 장?** (릴스 6장 / 캐러셀 7-9장)
- [ ] **예상 비용 확인** (장당 과금)

### 2단계: Reference 이미지 확인
- [ ] **인물 참조**: `assets/gena_hair (N).png` 포함 여부
- [ ] **제품 참조**: `assets/slant_*.png` 포함 여부 (제품 등장 시)
- [ ] **의상 참조**: `assets/outfit/` 사용 여부

### 3단계: 프롬프트 검증
- [ ] 프롬프트 파일 확인: `weekly/image-prompts.json`
- [ ] 각 슬라이드별 역할/목적 명확
- [ ] "no text, no watermark" 포함 여부
- [ ] 제품 디테일 (black nylon, side strap 등) 명시 여부

### 4단계: 최종 승인
- [ ] **마보스님 승인 받음** ✅
- [ ] 출력 경로 확인: `assets/output/W##-##-{format}/`
- [ ] 기존 이미지 덮어쓰기 여부 확인

## 생성 중 원칙

1. **1장씩 생성 → 확인 → 다음**
2. **중간 결과 보고** (2-3장마다)
3. **즉시 중단 가능** 상태 유지
4. **과금 누적 알림** (N장 생성됨)

## 생성 후 확인

- [ ] 생성된 이미지 개수 확인
- [ ] 인물 일관성 체크
- [ ] 제품 일관성 체크 (해당 시)
- [ ] 총 과금 비용 리포트
