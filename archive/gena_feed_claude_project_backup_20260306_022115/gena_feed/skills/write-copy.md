| 원칙 | 좋은 예 | 나쁜 예 |
|------|--------|--------|
| 가볍고 친근 | "이거 몰랐으면 진짜 억울했을 뻔" | "꼭 알아야 할 정보입니다" |
| 요점만 위트있게 | "결론: 순서가 전부임 😌" | "다양한 방법을 통해 개선할 수 있습니다" |

## 카피 공식

### 훅 (1장 / 릴스 S1)
- "[숫자]가지만 알면 됨"
- "이거 하나면 [문제 해결]"
- "아무도 안 알려준 [주제]"
- "[상황] 해본 사람 손"
- "나만 몰랐나 진짜로"

### 공감 (2장 / 릴스 S2)
- 상황 묘사 → "이거 나만 그런 거 아니지?"
- 1인칭 경험담으로

### 팁 (3~6장 / 릴스 S3~S4)
- `Tip N. [핵심 한 줄]`
- 부연 1~2줄 (구어체, 경험담)

### slant 자연 연결 (7장 / 릴스 S6 — monetize)
```
기본: "요즘 내가 [상황]에 들고 다니는 거"
     "이게 생각보다 [의외의 장점]이더라"

slant: "슬링백인데 진짜 아무데나 들어감 (링크바이오 🔗)"
파트너스: "[제품명] 써봤는데 이거 맞음 (링크바이오)"
```

L3 광고/협찬이면 반드시 1회: `#ad` 또는 `[브랜드명] 협찬`

### CTA (9장 / 릴스 S6 마지막)
- "저장해두고 나중에 써봐요 🗒"
- "링크바이오에서 바로 볼 수 있어요"
- "팔로우하면 이런 거 계속 올려요"

### 스토리 텍스트 (3종)
- D-1: "[주제 관련 질문 1줄]" + 투표 선택지 2개
- D+0: "오늘 이거 올렸어요 →" + 캐러셀 제목
- D+3: "[투표 결과]% 가 [선택]을 골랐어요 😄" + "링크바이오에서 바로 볼 수 있어요"
