# monetize SKILL

## 수익 레이어 우선순위

L1 → L2 → L3 순서로 우선 적용

### L1 — slant 자체 제품 (최우선)

노출 위치: 캐러셀 7장 · 스토리 D+3 링크 스티커 · 릴스 S6

카피 공식:
```
"요즘 어디 갈 때든 이거 들고 나옴"
"슬링백인데 생각보다 많이 들어감 (링크바이오 🔗)"
"가볍고 엣지 있어서 요즘 애정템 🫶 (링크바이오)"
```

- 광고 표기 불필요 (자체 제품)
- "(링크바이오)" 반드시 포함
- **주 1회 이상 반드시 노출**

### L2 — 파트너스 (제휴 링크)

노출 위치: 캐러셀 7장 or 릴스 S4

카피 공식:
```
"[제품명] 써봤는데 이거 맞음"
"요즘 이거 쓰고 있는데 진짜임 (링크바이오)"
```

- 광고 표기 불필요 (제휴 링크 고지는 바이오에서)

### L3 — 제휴 광고·협찬

노출 위치: 캐러셀 7장 or 릴스 S4 중반

```
표기 방법 (둘 중 하나):
  · 본문: "[브랜드명] 협찬"
  · 해시태그: #ad
위치: 해당 슬라이드 하단 or 릴스 자막 끝
```

- **반드시 1회 삽입 (누락 시 QA 고 이슈)**
- 광고 느낌 최소화: 주제와 맥락 연결 시에만 삽입
